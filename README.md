# APSRTC-Vizag_City_Bus_Fleet_Analysis
APSRTC Vizag City Bus Fleet Analysis. The Analysis is done taking APSRTC Yearly RTI into consideration. RTI  obtained contain data of APSRTC's total state fleet but not limited to Visakhapatnam. As a result most important parameters are deduced from RTI, Internet &amp; Levelled them down to Visakhapatnam City.
